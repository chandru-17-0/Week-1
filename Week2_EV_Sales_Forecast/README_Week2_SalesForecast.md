# Week-2 EV Sales Forecasting (Time-Series)

## Objective:
Forecast EV sales from 2025 to 2035 using Machine Learning (ARIMA + Linear Regression fallback).

## Steps:
- ✔ Used “Electric_Vehicle_Population_Data.csv”
- ✔ Extracted EV sales count per year based on “Model Year”
- ✔ Built baseline Linear Regression model
- ✔ Used Auto-ARIMA to generate future prediction
- ✔ Forecasted 2025 → 2035
- ✔ Exported prediction as CSV + plotted graph

- `EV_Sales_Forecasting_Week2.ipynb` (notebook)
